# Portfolio – Fiona #

Bienvenue sur mon portfolio !  
Je suis Fiona, étudiante en Informatique (Bac+3) à l’Université d’Évry Paris‑Saclay.  
Passionnée par le développement web, j’aime concevoir des sites assez modernes, intuitifs et agréables à utiliser.
Ce portfolio présente mes projets, mes compétences et mon profil.

---

- Aperçu du site

J'ai conçu ce site autour d'un thème : les saisons, pour structurer le site visuellement et donner du sens à la navigation.
J'ai utilisé une palette pastel propre à chaque saison et quelques animations douces en CSS, chaque page possède donc sa propre ambiance.
Ce choix reflète mon identité visuelle et mon intérêt pour les interfaces
esthétiques et équilibrées.
Le site a été pensé pour être lisible sur tous les écrans.
